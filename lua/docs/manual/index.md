# ezOS Manual

Welcome to ezOS, the operating system for the LilyGo T-Deck Plus.

This is the on-device manual. Pages here ship in firmware so they
work even with no SD card and no network.

## Sections

- Getting started
- Key bindings
- Mesh basics
- Maps
- Settings
- Customization (advanced)

Open any page from the Help app's main list. Use Backspace or Escape
to go back.

## More documentation

The full Lua API reference is too large to ship in firmware. Generate
it on a host with `python tools/generate_lua_docs.py` and copy the
output to `/sd/docs/api/` if you want it on the device.
