# Key bindings

Most screens follow the same conventions. App-specific keys are
listed in each app's section.

## Global

- Arrows / trackball: navigate menus and lists
- Enter: select / confirm
- Backspace or Escape: go back
- Q: quit the current screen (alias for back)
- LShift+RShift: open the app menu from any screen

## Lists and forms

- Up / Down: move focus
- Left / Right: switch between fields where applicable
- Enter: activate the focused row or button
- M: open the actions menu for the focused item (file manager, maps)

## Map

- Arrows: pan
- + / -: zoom in / out (also PgUp / PgDn)
- L: toggle labels
- T: toggle dark / light theme (legacy; the same toggle lives in
  Settings -> Display)
- G: jump to GPS fix; press again to follow
- H: jump once to GPS fix without follow
